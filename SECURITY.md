# Security Policy (SECURITY.md)

This security policy applies strictly to the public **Agentic AI Project Showcase** documentation repository.

---

## 1. Scope of This Repository
This repository is a **technical showcase and architecture portfolio**. It contains:
* High-level architectural guides and documentation.
* Mock configurations, sanitized metrics, and redacted workflow logs.
* Mermaid-based conceptual system diagrams.

**This repository does not contain production source code, executable binaries, deployment credentials, API keys, private system prompts, or live service endpoints.**

---

## 2. Reporting Vulnerabilities
Because this repository contains only public documentation and conceptual guides:
* **No Production Assets Here**: Do not file security issues or vulnerability reports against this public documentation repository.
* **Private Review Requests**: Serious security researchers, enterprise partners, or clients who wish to review the security architecture or conduct a private audit under MNDA can request access by contacting us directly.
