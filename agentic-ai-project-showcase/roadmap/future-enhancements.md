# Strategic Future Enhancements

This roadmap details the planned features, framework expansions, and architectural upgrades scheduled for future development cycles.

---

## Q3 2026: Active API Vulnerability Scanners
* **REST & GraphQL Fuzzing Workers**: Introduce deep API parameter fuzzing tools capable of testing rate limits, parameter manipulation, and authorization bypasses (BOLA/IDOR).
* **Automated API Spec Parsing**: Support direct ingestion of OpenAPI/Swagger schemas to dynamically map and test entire API surface areas.

---

## Q4 2026: Multi-Tenant Enterprise GRC Integration
* **Direct GRC Platform Sync**: Build native webhook connectors to automatically push signed evidence reports and findings to GRC dashboards like Drata, Vanta, and OneTrust.
* **Custom Compliance Frameworks**: Enable organizations to upload proprietary corporate security policies, automatically compiling findings mapped to internal company controls.

---

## Q1 2027: Native LLM Defense Middleware (Firewall)
* **Real-Time Guardrail Agent**: Package our prompt-isolation and sanitization designs into an active, high-performance API middleware module (LLM Firewall).
* **Adversarial Input Filtering**: Intercept and neutralize cognitive injection patterns before they reach core model systems, protecting applications in real time.
